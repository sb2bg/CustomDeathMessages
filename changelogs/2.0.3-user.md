# CustomDeathMessages 2.0.3 User Changelog

## Highlights

- Fixed item-hover death messages on modern Paper and Spigot versions.
- Added `/cdm debug hover [player]` so hover-item rendering can be tested without needing a real death event.
- Improved hover-item compatibility across legacy and modern server versions.

## Notes

- No config migration is required from `2.0.2`.
- This release is a patch update focused on hover-item stability and easier testing.
