# CustomDeathMessages 2.0.3 Developer Changelog

Changes since `248c584`

## Hover Item Compatibility

- Fixed hover-item chat rendering on modern Paper/Spigot by replacing the legacy raw-text `SHOW_ITEM` payload path with the structured hover item format when the newer Bungee chat API is available.
- Kept a reflection-based fallback to the older `HoverEvent(Action, BaseComponent[])` constructor so legacy servers can still use the previous hover-item path.
- Kept the existing NBT-based item hover source, but moved modern-server compatibility handling into `ItemSerializer` instead of relying on the command/debug layer.

## Commands and Testing

- Added `/cdm debug hover [player]` for testing hover-item rendering without needing to trigger an actual death event.
- Reused the real placeholder and hover transform pipeline in the debug command so it exercises the same path as live death messages.
- Kept the command gated behind `developer-mode` and `cdm.debug`.

## Release Metadata

- Bumped the released plugin version to `2.0.3`.
- Updated README command documentation to include the new hover debug command.
